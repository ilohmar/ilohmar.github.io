Emacs Window Configs with wconf.el   

Here’s a quick one that’s been on my to-do list for too long.

If you use Emacs for a lot of tasks, you might want to quickly switch between different window setups (think of the “workspaces” that many desktop environments and even some IDEs have). And it can be helpful to store and restore such configs between different Emacs sessions.

There are several Emacs packages that do this in one way or another, but they never do things the way I like them, or they do far too much and are fairly bloated (IMO, of course). With a decently up-to-date Emacs, however, it turned out to be really easy to write a small package myself: wconf

So go check it out, it might be just what you’re looking for (or what you needed and just never knew it). The package is now available via the official (pre-configured) GNU ELPA. It’s also very straight-forward to set up and get going. Have fun!
